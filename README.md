## Project Description
The **Shifra Training** project is a front-end educational website built using **HTML, CSS, and JavaScript**.  
The site includes multiple pages, such as:  
- Home / Explore Page  
- Courses Page  
- Course Detail Page  
- About Us Page
-Contact Page

The website is fully **responsive** and works seamlessly on desktop, tablet, and mobile devices.

---

## Project Files
1. `index.html` : The main HTML file of the website  
2. `/css/styles.css` : CSS file containing all styles and responsive rules  
3. `/js/main.js` : JavaScript file managing page navigation (`showPage`) and rendering courses dynamically (`renderCourses`)  
4. `/images/` : Folder containing all images used in the website  

---

## How to Run the Project
1. Ensure all files are organized as follows:
-  index.html
-  css/
-  styles.css
-  js/
- main.js
- images/
- all project images
2. Open `index.html` in any modern web browser (Chrome, Firefox, Edge).  
3. Use the **Navbar** to navigate between pages.  
4. Browse courses, filter by category, or click **"View Course"** to see course details.

---

## Project Features
- **Responsive Design:** Works on desktop, tablet, and mobile devices  
- **Dynamic Courses Display:** Courses are rendered dynamically based on category using JavaScript  
- **Interactive Navbar:** Toggleable menu on mobile devices  
- **Course Detail Page:** Includes detailed course information, description, price, features, and student reviews

---

## Mobile Access
- You can host the project on **GitHub Pages** or open `index.html` directly in a mobile browser.  
- Ensure CSS and JS files are properly linked in the HTML.
